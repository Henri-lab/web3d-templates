# 首页模块 (module-welcome)

> 平台首页和导航入口模块

## 📖 模块概述

首页模块是平台的入口页面，提供了模块导航、平台介绍和快速访问功能。

### 核心功能

- ✅ **模块导航** - 展示所有可用模块
- ✅ **快速访问** - 快速跳转到常用模块
- ✅ **平台介绍** - 展示平台功能和特性
- ✅ **动画效果** - 流畅的页面动画
- ✅ **响应式设计** - 适配不同屏幕尺寸

## 🎯 模块信息

| 属性 | 值 |
|------|-----|
| 模块 ID | `welcome` |
| 模块名称 | 首页 |
| 版本 | 1.0.0 |
| 类型 | local |
| 命名空间 | `welcome` |

## 🚀 快速开始

### 访问模块

```
http://localhost:5173/
```

### 模块配置

```typescript
{
  id: 'welcome',
  name: '首页',
  description: '平台首页，模块导航入口',
  version: '1.0.0',
  type: 'local',

  routes: [
    {
      path: '/',
      component: 'WelcomePage',
      meta: { title: '首页' },
    },
  ],

  menu: {
    title: '首页',
    icon: 'home',
    order: 0,
    visible: true,
  },
}
```
